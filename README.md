# Nework_Bandwidth_Monitor


SNMP based network bandwidth , data usage and Speed monitor for SNMP enabled devices

## Requirements
   SNMP Enabled devices/interfaces
   
   MySql Database
   
   PDF reader to view reports

# How to use
Clone the repo and open in netbeans

# Dependencies
   - SNMP4J
   - iText
